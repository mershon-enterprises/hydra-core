Login and Logout

Logging in:
To begin using the Hydra File Explorer, enter your Email and Password on the login page and click Login.

Logging out:
When ready to logout of your account, click the Logout button on the right side of the top menu bar.

Upload and Download Files

Upload a file:
Access the Upload File page through the link on left side of the File Explorer menu bar.

Continue by clicking Choose File and navigating to the file you wish to upload. Once selected, the file information will be displayed with options to rename and tag the file before upload. Rename the file by typing in the Filename field where the current filename is displayed. Add tags by filling out the empty Tag Name and Tag Value fields and clicking the icon, and remove tags by clicking the icon. When desired changes are complete click Save to finish, or cancel by clicking the Back button.

Download a file:
A copy of a file can be downloaded from the File Explorer to your local hard drive using the associated icon on the right side of the file list.

View and Edit Files

View file details:
File details can be accessed from the file list by clicking anywhere in the associated file row or by clicking the icon to the right. Use the Back button or the File Explorer link to return to the file list.

Replacing a file:
Use the Replace File button on the file details page and navigate to the new file you wish to overwrite the current file with. Replacing a file will maintain the original filename and tags.

Editing file tags:
Add tags on the file details page by filling out the empty Tag Name and Tag Value fields and clicking the icon, and remove tags by clicking the icon.

Renaming a file:
A filename can be renamed on the file details page by entering text in the lower filename field and clicking the Rename File button.

Search and Navigation

Search files:
You can search through files by Filename, Author, Creation Date, and Tag Value. Reset the file list to display all with a blank search or by clicking the Reset button from the navigation options.

Searching for files by Creation Date the search must be part of the following format: [YYYY]-[MM]-[DD]

Navigating pages:
Above the file list are options for display and navigation of pages. The 25, 50, and 100 options will change the number of entries displayed per page. Use Reset to return to default display settings. Navigate between pages with the arrows on the right. Single arrows ( ) change one page at a time, double arrows ( ) jump to the first or last page, and the drop down menu can select a specific page number.

Sorting file list:
Click a the name of a file attribute (Ex: "Author") to toggle between descending () and ascending () sorting of the file list. Collapse categories by clicking the icon and expand by clicking the icon.
