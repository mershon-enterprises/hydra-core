Login and Logout

Logging In

To begin using the Hydra File Explorer, enter your Email and Password on the login page and click Login.

Logging Out

When ready to logout of your account, click the Logout button on the right side of the top menu bar.

Upload and Download Files

Upload a File

Access the Upload File page through the link on left side of the File Explorer menu bar (keyboard shortcut: Alt-U).

Continue by clicking Choose File (keyboard shortcut: Alt-C) and navigating to the file you wish to upload. Once selected, the file information will be displayed with options to rename and tag the file before upload. Rename the file by typing in the Filename field where the current filename is displayed. Add tags by filling out the empty Tag Name and Tag Value fields and clicking the icon, and remove tags by clicking the icon. When desired changes are complete click Save to finish, or cancel by clicking the Back button.

For easy entry of tag information, use the Tab key to switch between tag name and value fields. Using Tab again while in the value field will automatically select and create a new tag row.

Download a File

A copy of a file can be downloaded from the File Explorer to your local hard drive using the associated icon on the right side of the file list, as well as the green Download button on the file details page as seen in the following section.

View and Edit Files

View File Details

File details can be accessed from the file list by clicking anywhere in the associated file row or by clicking the icon to the right. Use the Back button or the File Explorer (keyoard shortcut: Alt-F) link to return to the file list.

Renaming a File

A filename can be renamed on the file details page by entering text in Rename field and clicking the Rename button.

Sharing a File

A file can be shared from the file details page by clicking the Share button. This will generate a link that can be copied and sent to those you wish to share the file with. Each link will have a default expiration one week from the time it was generated. You can change the expiration date before generating a link by clicking the Expiration Date text and selecting from the calendar.

Editing File Tags

Add tags on the file details page by filling out the empty Tag Name and Tag Value fields and clicking the icon, and remove tags by clicking the icon.

For easy entry of tag information, use the Tab key to switch between tag name and value fields. Using Tab again while in the value field will automatically select and create a new tag row.

Replacing a File

Replace functions as an alternative to deleting and uploading a new file but will maintain the currently entered filename and tags. Use the Replace button on the file details page and navigate to the new file you wish to overwrite the current file with.

Search and Navigation

Searching Files

You can search files with the search field above the file list (keyboard shortcut: / ). You may serch by Filename, Author, Creation Date, and Tag Value. Reset the file list to display all files by performing a blank search (keyboard shortcut: Esc) or by clicking the Reset button from the navigation options.

Searching for files by Creation Date the search must be part of the following format: [YYYY]-[MM]-[DD]
Examples: Search for entries on January 3rd, 2015 by entering "2015-01-03‚Äù as a search without quotes. Search for all January entries with "2015-01"

Navigating and Sorting File List

At the top and bottom of the file list a navigation bar provides options for display and navigation of pages. The 25, 50, and 100 options will change the number of entries displayed per page. Use Reset to return to default display settings. Navigate between pages with the arrows on the right. Single arrows change one page at a time, double arrows jump to the first or last page, and the drop down menu can select a specific page number.

Click a the name of a file attribute (Filename, File Size, Author, Creation Date) to toggle between descending and ascending sorting of the file list. Collapse categories by clicking the icon and expand by clicking the icon.
